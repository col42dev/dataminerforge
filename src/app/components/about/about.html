Dataminer imports from google spreadsheets, converts to runtime ready JSON format and exports to AWS DynamoDB. Each exporter targets live and test tables in DynamoDB, exporting to the live channel will also export to the test channel and to the Perforce depot. The Dataminer Backfiller app manages syncing the Perforce depot from the DynamoDB live table. The output for this sync is logged.

To generate a new dynamic data set for Dataminer, name the work sheet starting with an underscore. The Dynamic Worksheets page will locate these worksheets and provdes a generic exporter for them.
